# Vi-MTPro Cascade

Быстрая установка MTProxy с поддержкой каскада через WARP.

Два режима работы:
- **Одиночный** — один VPS, mtg подключается к Telegram напрямую
- **Каскад** — RU-сервер → EU-сервер → WARP → Telegram (обход блокировок)

---

## Быстрый старт

### RU-сервер (основной)

```bash
bash <(curl -sSL https://raw.githubusercontent.com/ViPunch/Vi-MTProxy/master/setup.sh)
```

При первом запуске выберите режим — одиночный или каскад. Скрипт установит mtg, создаст первого клиента и выдаст ссылку для Telegram.

---

### EU-сервер (только для режима каскада)

```bash
bash <(curl -sSL https://raw.githubusercontent.com/ViPunch/Vi-MTProxy/master/tunnel.sh)
```

Скрипт установит WARP и gost, поднимет SOCKS5-туннель на порту 1080. После этого вернитесь на RU-сервер — каскад заработает.

---

## Что умеет setup.sh

- Добавлять неограниченное количество клиентов (каждый на своём порту и SNI)
- Показывать готовые `tg://` ссылки для всех клиентов
- Привязывать и отвязывать EU-сервер без пересоздания клиентов
- Обновлять mtg до последней версии
- Удалять отдельных клиентов или всё целиком

## Что умеет tunnel.sh

- Устанавливать WARP и gost одной командой
- Показывать статус туннеля
- Полностью удалять туннель

---

## Требования

- Ubuntu 24.04
- Root-доступ
- Открытый порт 443 (или любой другой по выбору) на RU-сервере
- Открытый порт 1080 на EU-сервере (только для каскада)
